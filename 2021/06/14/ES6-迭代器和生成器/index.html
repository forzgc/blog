ES6 生成器和迭代器

生成器和迭代器

迭代器是一个特殊对象，有一个next()方法，每次调用迭代器的next方法都会返回一个结果对象：

生成器是可以返回迭代器的函数，特点是函数名前带符号：*，（匿名函数也要带上）。还可以使用关键字yield，这个关键字的作用是调用迭代器的next方法后返回一个值。

yield的特点是每次迭代器next方法调用后，生成器函数执行到yield会返回结果对象并暂停，如果函数后面没有yield，那么生成器函数就会关闭并返回：

可迭代对象

可迭代对象是指具有Symbol.interator属性的对象，调用这个属性会返回一个关于这个对象的迭代器。
ES6中所有集合对象（Array Set Map）以及 String 都有这个属性，都是可迭代对象。

for-of

for-of是一个可对迭代对象进行循环的语法。
使用for-of时，javascript引擎会调用可迭代对象的Symbol.interator属性，得到一个迭代器。调用迭代器的next方法得到结果对象，然后将value值传出并执行for-of{}中的代码，直到返回结果对象中done为true时退出循环。

默认迭代器

可迭代对象的默认迭代器由Symbol.interator属性指定
普通对象不是可迭代对象，可通过创建一个Symbol.interator属性并赋值为一个生成器，来变为一个可迭代对象。

内建迭代器

集合对象(Array Set Map)的内建迭代器有三种

Map的默认迭代器是 entries()
Array和Set的默认迭代器是 values()

可迭代对象的Symbol.iterator、entries、values和keys属性类似生成器。
