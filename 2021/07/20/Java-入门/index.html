Java 入门

main入口函数

main入口函数有以下要求：

  • 必须使用 public 修饰符
  • 必须使用 static 修饰符
  • 必须有一个String数组的参数
  • 返回类型是void

java语言大小写敏感

Java源文件结构

  • 零个或一个包声明
  • 零个或多个包引入
  • 零个或多个类声明

包声明位于java文件代码顶部，一个java文件只能有一个包声明语句，如果没有声明，那么就在默认包中。

如果一个Java源文件有多个类，那么最多只有一个类是public，并且这个类名称必须和源文件名称相同。
